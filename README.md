## N Body Simulation

## Startup / Dev

1. `yarn`
2. `yarn watch`
3. open `localhost:3000`

## Build
1. `yarn`
2. `yarn build`
3. look at `dist` dir

## About

This is "N Body Simulation". Specifically, the collisions and gravity of particles in a limited area are simulated. The UI has simulation settings.
